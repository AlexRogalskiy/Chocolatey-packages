The newest build of the individual, Oswald font files can be 
directly downloaded by digging into the GitHub repository:

https://github.com/googlefonts/OswaldFont/tree/main/fonts/otf

Click on each font and then click the "download" button.

A potentially easier method is to download the zipped repository
(https://github.com/googlefonts/OswaldFont/archive/refs/heads/main.zip)
and then unzip and extract the files in .\fonts\otf.
